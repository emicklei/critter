Conditions

Host

<host matches="www.mine.com" />
True if the hostname of the request matches the given value.

Port

<port matches="80" />
True if the URL port of the request matches the given value.

Path

<path matches="/order/*" />
True if the URL path of the request matches the given value. Use can use capturing groups , e.g. /products/(.+) and use its value in the response body by referring them as $path.1$ for the first group

Method

<method matches="GET" />
Only true if the Http method is equal to (and matches) GET.

RequestHeader

<requestheader name="Content-Type" matches="application/xml" />
True if the specified Request Header (by name) of the request matches the given value.

RequestBody

<requestbody matches="*"" />
True if the specified Request Body (content) of the request matches the given value.

BasicAuthentication

<basicauthentication username="Aladdin" password="open sesame" />
True if the Authorization header matches the Base64 encoded version of username:password combination

Not

<not>
    <host matches="not.test.critter" />
</not>
Any condition can be wrapped in a not element to negate it.
